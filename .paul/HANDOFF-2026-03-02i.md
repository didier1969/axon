## Architecture (daemon)

```
~/.axon/daemon.sock   ← asyncio Unix socket (JSON-line protocol)
~/.axon/daemon.pid    ← PID file

axon daemon start     → subprocess.Popen([python, -m, axon.daemon, --max-dbs, 5])
axon daemon stop      → os.kill(pid, SIGTERM)
axon daemon status    → reads PID + queries daemon via socket for cache stats

Daemon dispatch:
  Request:  {"id":"1","tool":"axon_query","slug":"myapp","args":{...}}\n
  Response: {"id":"1","result":"...","error":null}\n

LRU cache: dict(slug → KuzuBackend), maxsize=5, thread-safe OrderedDict
```

**Note:** MCP still uses direct KuzuBackend (unchanged). Plan 02-02 makes MCP a proxy to the daemon.
