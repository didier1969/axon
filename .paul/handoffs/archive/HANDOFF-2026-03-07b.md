## Decisions Made

| Decision | Rationale | Impact |
|----------|-----------|--------|
| `default to` language (not `always`) for axon routing | Grep has legitimate uses for literals, regex, unknown files | Balanced guidance — agents use axon for symbols, grep for strings |
| `axon_summarize` added to routing for analysis | Agents reading files directly instead of using axon for "analyze" requests | Now triggers axon_summarize for file/module analysis |
| Daemon evict before analyze (not kill MCP) | KuzuDB exclusive lock even in read-only mode; killing MCP is disruptive | `axon analyze` gracefully closes daemon connection before writing |
| systemd user service for watchers | "always active" goal requires persistent process beyond terminal lifetime | `axon-watch@trader-elixir-v2` etc. post WSL restart |
| `~/.gemini/GEMINI.md` for Gemini instructions | Confirmed in Gemini CLI source: this path is the global memory/context file | Gemini loads axon routing instructions at startup |
