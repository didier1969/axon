## Architecture décidée dans le plan

```
Ancien (❌)                    v0.6 Phase 1 (✅)
{project}/.axon/kuzu           ~/.axon/repos/{slug}/kuzu
{project}/.axon/meta.json      {project}/.axon/meta.json (+ "slug" field)
```

**Slug** : `{repo_name}` ou `{repo_name}-{sha256[:8]}` en cas de collision (même logique qu'avant dans `_register_in_global_registry`).

**Migration automatique** : dans `analyze`, si `{project}/.axon/kuzu` existe et `~/.axon/repos/{slug}/kuzu` n'existe pas → `shutil.move`.

**Backward compat** : si `meta.json` n'a pas de champ `slug` → fallback sur `{project}/.axon/kuzu`.
