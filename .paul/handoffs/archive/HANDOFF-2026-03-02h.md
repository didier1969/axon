## Architecture (Plan 02-01)

```
~/.axon/daemon.sock   ← Unix socket (asyncio, JSON-line protocol)
~/.axon/daemon.pid    ← PID file

axon daemon start     → subprocess.Popen([python, -m, axon.daemon, --max-dbs, 5], start_new_session=True)
axon daemon stop      → os.kill(pid, SIGTERM)
axon daemon status    → reads PID + queries daemon via socket for cache stats

Daemon dispatch:
  Request:  {"id":"1","tool":"axon_query","slug":"myapp","args":{...}}\n
  Response: {"id":"1","result":"...","error":null}\n

LRU cache: dict(slug → KuzuBackend), maxsize=5, evict LRU on overflow (calls close())
```
