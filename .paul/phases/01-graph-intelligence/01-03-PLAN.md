<objective>
## Goal
Add three intelligence capabilities: (1) code-aware embedding text that includes actual source content for better semantic search, (2) a new `axon_find_similar` MCP tool for semantic duplicate detection, (3) optional query expansion and surface `tested`, `is_exported`, `centrality` in MCP output.

## Purpose
Embeddings built from metadata-only text miss semantic intent in the code body. Including the actual source snippet improves vector search quality and makes `axon_find_similar` more useful. Surfacing `tested` and `is_exported` in MCP responses turns axon into an actionable code quality tool — agents can identify untested, unexported dead code without writing Cypher.

## Output
- Modified: embeddings/text.py, mcp/tools.py, mcp/server.py
- New tests: test_tools.py (find_similar), test_text.py (content inclusion)
</objective>
