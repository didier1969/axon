<objective>
## Goal
Add Elixir, Rust, and Markdown language support to the Axon parser pipeline, making all developer projects parseable by the indexer.

## Purpose
Axon can only deliver code intelligence for languages it parses. Without Elixir, Rust, and Markdown support, a large portion of the developer's projects (Elixir OTP apps, Rust tools, documentation) are invisible to AI agents querying the index.

## Output
- `elixir_lang.py` — tree-sitter Elixir parser extracting modules, functions, macros, structs, imports, heritage, calls
- `rust_lang.py` — tree-sitter Rust parser extracting functions, structs, enums, traits, impls, mods, type aliases, use declarations, calls
- `markdown.py` — parser extracting headings as sections, links, and code block calls
- All parsers registered in `__init__.py`, `languages.py`, `parser_phase.py`
- 81 new tests (Elixir: ~30, Rust: ~38, Markdown: ~13)
- Full suite: 645 tests passing
</objective>
