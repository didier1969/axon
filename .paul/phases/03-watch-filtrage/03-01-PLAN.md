<objective>
## Goal
Add `.paul/` to built-in ignore patterns, add a watchfiles pre-filter so ignored paths never enter the watcher pipeline, and make debounce configurable via a parameter and CLI flags.

## Purpose
The watcher currently fires on `.paul/` directory changes (PAUL handoff/state edits), wasting reindex cycles. Debounce is hardcoded at 500ms — users with slow machines or fast CI environments need to tune this. These are independent polish items with no schema or queue changes.

## Output
- `DEFAULT_IGNORE_PATTERNS` gains `.paul/`
- `watch_repo()` gains `debounce_ms: int = 500` and a `watch_filter` that pre-filters ignored paths
- `axon watch` and `axon serve --watch` gain a `--debounce INTEGER` option
- Tests confirm `.paul/` is ignored and debounce param is accepted
</objective>
