<objective>
## Goal
Two focused quality improvements: (1) skip files >512KB in walker.py to prevent OOM on large repos; (2) extract the slug computation logic from 3 duplicated inline blocks in cli/main.py into a `compute_repo_slug()` helper in `core/paths.py`.

## Purpose
Walker currently reads all files into memory regardless of size — a file >1MB can cause OOM during indexing. Slug computation is duplicated verbatim at 3 call sites in cli/main.py with identical logic and a shared comment "mirrors _register_in_global_registry" — a clear DRY violation.

## Output
- `walker.py`: `read_file()` returns `None` with a `logger.warning` for files >512KB
- `core/paths.py`: new `compute_repo_slug(repo_path: Path) -> str` function
- `cli/main.py`: 3 inline slug blocks replaced with `compute_repo_slug(repo_path)`
- `tests/core/test_walker.py`: `TestWalkRepoSkipsLargeFiles` class added
</objective>

  <action>
    Add `compute_repo_slug(repo_path: Path) -> str` to `core/paths.py`:
    - Add `import hashlib`, `import json` at the top of paths.py
    - Implement the exact same logic as the 3 inline blocks:
      ```python
      def compute_repo_slug(repo_path: Path) -> str:
          """Compute the registry slug for a repo path.

          Returns repo_path.name unless that name is already claimed by a
          different repo in the global registry, in which case returns
          f"{repo_path.name}-{sha256(str(repo_path))[:8]}".
          """
          slug = repo_path.name
          candidate_meta = Path.home() / ".axon" / "repos" / slug / "meta.json"
          if candidate_meta.exists():
              try:
                  existing = json.loads(candidate_meta.read_text(encoding="utf-8"))
                  if existing.get("path") != str(repo_path):
                      slug = f"{repo_path.name}-{hashlib.sha256(str(repo_path).encode()).hexdigest()[:8]}"
              except (json.JSONDecodeError, OSError):
                  pass
          return slug
      ```
    - No side effects (no shutil.rmtree, no mkdir) — pure computation only
  </action>

    ```python
    class TestWalkRepoSkipsLargeFiles:
        """Files exceeding 512 KB are silently skipped with a warning."""

        def test_large_file_skipped(self, tmp_path: Path) -> None:
            # Create a .py file just over 512KB
            large_file = tmp_path / "huge.py"
            large_file.write_bytes(b"x = 1\n" * 90_000)  # ~540 KB

            # Create a normal .py alongside it
            small_file = tmp_path / "small.py"
            small_file.write_text("y = 2", encoding="utf-8")

            entries = walk_repo(tmp_path)
            paths = {e.path for e in entries}

            assert "huge.py" not in paths
            assert "small.py" in paths

        def test_large_file_triggers_warning(self, tmp_path: Path, caplog: pytest.LogCaptureFixture) -> None:
            import logging
            large_file = tmp_path / "big.py"
            large_file.write_bytes(b"# comment\n" * 60_000)  # ~600 KB

            with caplog.at_level(logging.WARNING, logger="axon.core.ingestion.walker"):
                walk_repo(tmp_path)

            assert any("big.py" in record.message for record in caplog.records)

        def test_file_exactly_at_limit_skipped(self, tmp_path: Path) -> None:
            # 512 * 1024 bytes exactly — should be skipped (strictly greater)
            boundary_file = tmp_path / "boundary.py"
            boundary_file.write_bytes(b"a" * (512 * 1024 + 1))

            entries = walk_repo(tmp_path)
            paths = {e.path for e in entries}
            assert "boundary.py" not in paths

        def test_file_just_under_limit_included(self, tmp_path: Path) -> None:
            under_file = tmp_path / "fine.py"
            under_file.write_bytes(b"x = 1\n" * 85_000)  # ~510 KB

            entries = walk_repo(tmp_path)
            paths = {e.path for e in entries}
            assert "fine.py" in paths
    ```
